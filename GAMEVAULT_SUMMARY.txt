 ASPEK KEAMANAN (Security Best Practices)
════════════════════════════════════════════════════════════════════════════════

Session Configuration:
├─ session.use_only_cookies = 1        (hanya cookie, bukan URL parameter)
├─ session.cookie_httponly = 1         (JS tidak bisa akses cookie)
├─ session.cookie_samesite = Strict    (CSRF protection)
└─ session_name('GAME_STORE_SESSION')  (custom session name)

CSRF Protection:
├─ Binary random token (32 bytes)
├─ Token di semua form POST (hidden input)
├─ Validasi dengan hash_equals() (timing-safe)
└─ Token regenerate setiap request

Input Validation:
├─ Type casting: (int) pada numeric inputs
├─ Cek product_id ada di database
├─ Validasi quantity (min 1, max = stok)
└─ htmlspecialchars() pada semua output (XSS prevention)

Data Integrity:
├─ Stok tidak bisa melebihi yang tersedia
├─ Validasi sebelum add/update quantity
├─ Error handling untuk missing products
└─ Prevent duplicate cart items
